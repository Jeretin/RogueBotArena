# RogueBotArena

## Description
This is a demo project. Game idea is to battle against enemy bots with gun and dashing ability.

Goal is to survive as long as possible and kill as many enemies as possible.


**Game engine:** Unity 6

**Type:** 3D Top-Down Shooter
